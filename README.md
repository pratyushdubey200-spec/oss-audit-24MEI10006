Project Overview

This project presents an in-depth evaluation of Git as an open-source software tool. It covers its background, licensing model, ecosystem, and a comparison with proprietary alternatives. In addition, the project includes practical work using Linux shell scripting to demonstrate hands-on knowledge of open-source utilities and system-level operations.

The main goal is to gain both conceptual understanding and practical experience with open-source software and recognize its role in today’s computing environment.

Script Summary

Script 1: System Information Display
Shows essential system details such as kernel version, current user, uptime, system date, and Linux distribution.

Script 2: FOSS Package Checker
Verifies whether Git is installed and provides information like its version and description.

Script 3: Storage and Permission Analyzer
Examines directories to report disk usage and file permissions.

Script 4: Log Analyzer Tool
Searches for a specific keyword in a log file and outputs matching entries along with occurrence count.

Script 5: Manifesto Creator
Generates a customized open-source manifesto based on user-provided input.

Execution Steps

Step 1: Grant Execute Permissions

chmod +x script1.sh
chmod +x script2.sh
chmod +x script3.sh
chmod +x script4.sh
chmod +x script5.sh

Step 2: Run the Scripts

./script1.sh
./script2.sh
./script3.sh
./script4.sh /var/log/syslog error
./script5.sh
System Requirements
Ubuntu (WSL) or any Linux-based environment
Bash shell
Git installed on the system
Output Details

All scripts display results directly in the terminal, while Script 5 additionally creates a text file containing the generated manifesto.
